# Feature Template

This directory is the canonical source for small feature-spec templates.

Use the templates as a feature bundle:

- `spec.example.md`: required feature-spec schema
- `task-plan.example.md`: implementation and review task split by spec id
- `openapi.example.yaml`: API contract example when the slice exposes an API
- `model.example.md`: domain and repository example notes
- `changelog.example.md`: optional change summary after implementation and review

Copy these templates into scaffolded projects. Do not evolve downstream copies independently from this directory.
